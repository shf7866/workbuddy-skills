# 潜在类别分析（LCA）方法论文献

## 一、理论基础

### 1.1 核心原理

潜在类别分析（Latent Class Analysis, LCA）是一种以**人为中心**的**概率性分类**方法。其核心假设是：观测变量之间的关联可以通过少数潜在类别来解释，同一类别内的个体在观测变量上具有相似的反应模式。

**数学形式**：
$$P(Y) = \sum_{k=1}^{K} \pi_k \prod_{j=1}^{J} \prod_{r=1}^{R_j} P(Y_{jr}|C_k)^{\mathbb{I}(Y_{jr})}$$

其中：
- $\pi_k$：类别概率（latent class probability），表示个体属于类别k的先验概率
- $P(Y_{jr}|C_k)$：条件概率，表示属于类别k的个体在观测变量j上选择响应r的条件概率
- $K$：潜在类别数量
- $Y$：观测变量矩阵

### 1.3 与传统方法的根本区别

| 特征 | LCA（以人为中心） | 因子分析（以变量为中心） |
|------|-------------------|--------------------------|
| 分析单元 | 个体/案例 | 变量/指标 |
| 输出 | 类别归属概率 | 因子得分 |
| 假设 | 类别间异质性 | 连续潜在变量 |
| 解释重点 | "什么样的人" | "什么因素" |

## 二、模型选择与识别

### 2.1 模型选择指标体系

| 指标 | 计算方式 | 判别标准 | 权重 |
|------|----------|----------|------|
| **BIC** | $-2LL + k \cdot \ln(n)$ | 越小越好，差异>10强烈支持 | ⭐⭐⭐⭐⭐ |
| **AIC** | $-2LL + 2k$ | 越小越好 | ⭐⭐⭐ |
| **aBIC** | 样本校正BIC | 越小越好，小样本首选 | ⭐⭐⭐⭐ |
| **Entropy** | 分类精度指标 | >0.80优秀，>0.60可接受 | ⭐⭐⭐⭐ |
| **LMR-LRT** | Lo-Mendell-Rubin检验 | p<0.05支持K个类别 | ⭐⭐⭐⭐ |
| **BLRT** | Bootstrap似然比检验 | p<0.05支持K个类别 | ⭐⭐⭐⭐ |
| **最小类别比例** | 各类别占比 | >5%实际意义 | ⭐⭐⭐⭐ |

**决策层级**：
1. **首选**：BIC最优（通常差异>10为强烈证据）
2. **验证**：Entropy > 0.80，类别可解释性
3. **检验**：LMR-LRT / BLRT p < 0.05
4. **底线**：最小类别比例 > 1%

### 2.2 分类质量标准

```
优秀：Entropy ≥ 0.90，分类误差极低
良好：Entropy ≥ 0.80，分类质量可靠
可接受：Entropy ≥ 0.60，需谨慎解释
不推荐：Entropy < 0.60，类别区分度不足
```

### 2.3 模型识别问题

**常见识别失败原因**：
1. **稀疏数据**：某类别条件概率为0或1
2. **局部独立假设违反**：类别内观测变量仍相关
3. **参数过多**：超过样本量承载能力
4. **起始值不良**：EM算法陷入局部最优

**解决策略**：
- 增加样本量或减少观测变量
- 合并稀有响应类别
- 多次随机起始值重启
- 检查观测变量相关结构

## 三、测量等值性检验

### 3.1 为什么需要等值性检验

LCA假设同一类别内的条件概率相同。但**跨组比较**时，必须先验证测量等值性（Measurement Invariance），否则类别比较可能产生误导。

### 3.2 检验序列

```
Configural Model（形态等值）
    ↓ LMR-LRT / BLRT (p < 0.05 → 拒绝)
Metric Model（弱等值）→ 所有组的条件概率差异仅限scale
    ↓ LMR-LRT / BLRT (p < 0.05 → 拒绝)
Scalar Model（强等值）→ 截距/ thresholds 完全相等
    ↓ LMR-LRT / BLRT (p < 0.05 → 拒绝)
```

### 3.3 等值性判断标准

| 模型 | BIC变化 | 解释 |
|------|---------|------|
| Configural → Metric | ΔBIC < 0 | 支持弱等值 |
| Metric → Scalar | ΔBIC < 0 | 支持强等值 |

**注意**：等值性检验在LCA中不如CFA严格，但跨组比较时强烈推荐。

## 四、软件实现

### 4.1 Mplus语法模板

```mplus
TITLE: LCA Model Specification;
DATA: FILE IS data.csv;
       VARIABLE: NAMES ARE y1-y5;
                CATEGORICAL = y1-y5;
                CLASSES = c(2);
ANALYSIS: TYPE = MIXTURE;
          ESTIMATOR = MLR;
          STARTS = 200 50;
          STITERATIONS = 20;
MODEL: %OVERALL%
         %c#1%
           [y1$1-y5$1] (thresholds);
         %c#2%
           [y1$1-y5$1];
OUTPUT: TECH1 TECH8;
SAVEDATA: FILE = lca_results.csv;
          SAVE = CPROB;
PLOT: TYPE = PLOT3;
      SERIES = y1-y5(c);
```

### 4.2 R实现（poLCA包）

```r
library(poLCA)

# 观测变量必须为因子型
data$gender <- factor(data$gender)
data$income <- factor(data$income)

# 模型公式
f <- cbind(y1, y2, y3, y4, y5) ~ 1

# 循环比较2-5个类别
results <- lapply(2:5, function(k) {
  poLCA(f, data, nclass = k, nrep = 10, 
        maxiter = 1000, verbose = FALSE)
})

# BIC比较
BICs <- sapply(results, function(x) x$bic)
optimal_k <- which.min(BICs) + 1
```

### 4.3 SAS实现（PROC LCA）

```sas
proc lac;
  title 'LCA with 3 classes';
  class y1-y5;
  dist y1-y5 = multinomial;
  parms y1-y5;
  numcat y1=2 y2=3 y3=2 y4=2 y5=3;
  classes c(3);
  run;
```

## 六、LCA在HR/教育/心理领域的应用

| 领域 | 研究问题 | 观测变量类型 |

## 七、常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 模型不收敛 | 类别过多样本不足 | 减少K或变量 |
| 局部最优 | 起始值不当 | 增加STARTS次数 |
| 类别不可解释 | 模型错误 | 重新审视变量选择 |
| 类别比例悬殊 | 类别内异质性高 | 细分或合并类别 |
| 条件概率接近 | 类别区分度低 | 减少K |

## 八、方法论张力与争议

### 8.1 本质主义 vs 建构论

**批评者观点**：
- 潜在类别可能只是统计人工物
- "真实存在"的类别是本质主义假设
- 建构论者认为类别是研究者的建构

**回应**：
- LCA是描述性工具，非本体论主张
- 强调类别的"实用有效性"而非"形而上学真实性"
- 结合定性研究验证类别意义

### 8.2 样本依赖性

- 不同样本可能产生不同最优K
- 小样本（<200）类别估计不稳定
- 建议进行稳健性检验

### 8.3 类别数的人为性

- 连续→离散的转化必然损失信息
- 决策树提供帮助但不能完全客观
- 最终需要理论判断
